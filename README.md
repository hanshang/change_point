# change_point
Change-point detection in functional time series: Applications to age-specific mortality and fertility 
1. AUS_fertility_data.R: R code for identifying change points for the age-specific Australian fertility
2. AUS_mortality_data.R: R code for identifying change points for the age- and sex-specific Australian mortality
3. Fert_AUS.csv: data file containing the latest Australian fertility rates
4. GK_test.R: independence test for functional time series
5. change_FF.R: fully functional change-point detection method
6. change_point_fChange_AUS_fert.RData: RData environment for the age-specific fertility
7. change_point_fChange_AUS_mort.RData: RData environment for the age- and sex-specific mortality
8. change_regression.R: regression-based approach for identifying change points
9. load_packages.R: load R packages
10. long_run_covariance_h.R: estimate long-run covariance function by kernel sandwich estimator
11. save_function.R: R code for saving figures
